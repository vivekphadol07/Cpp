# Bit Manipulation  
- Bitwise AND -> &      
- Bitwise OR  -> |      
- Bitwise XOR -> ^      

**One's Not Operator :**        
- Binary Not Operator ( ~ )     

**Binary Shift Operators :**
- Left Shift ( << ) // a << b -> a * 2^b          
- Right Shift ( >> ) // a >> b -> a / 2^b       

**Bitmask :**   
-Bitmask are used to access specific bits in byte od data.
