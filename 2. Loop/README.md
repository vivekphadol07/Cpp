# Loop

## 1. for Loop
- **Syntax**                      
                                                      
        for( initialisation ; condition ; updation ){
      
            // some work
        }                          
                
## 2. while Loop
- **Syntax**      
                                                   
        // initialisation    
        while ( condition ){  
          
                //Updation  

        }                      
                                
## 3. do while Loop
- **Syntax**
            
      do{

              //do some work

      } while( condition );
  
## 4. Break Statement    
- To exit the loop.      

## 5. Continue Statement    
- to skip an iteration    

## 6. Nested Loop    
- Loop inside Loop    
- **Outer loop:** No. of rows.  
- **Inner loop:** No. of columns
